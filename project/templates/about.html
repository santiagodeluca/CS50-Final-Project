    About this project

    This website is my final project for the CS50 course! The idea behind this was to create a site where people who are learning english can have a quick way to see an categorize the words on a text, and learn them. This was something I always felt like I would love to use if it existed, mainly when studying long texts and wanting to learn its vocab!

    References

    When trying to find way to determine wheter a word in enlish had a high or low complexity level I thought a great to do so would be to use a list of frequently used words. Luckily, the github user Hermit Dave has popular repository called FrequencyWords where there are multiple lists of words order by frequency of use in multiple languages. One of the is a 50,000 word list in english, which is the database I used as the center of this page.
